# ⚡ LinkedIn Easy Apply Bot

Automatically applies to LinkedIn Easy Apply jobs that match your criteria — no clicking, no copy-pasting, no babysitting.

> Built by a friend. Share freely.

---

## Install in 3 steps

### Step 1 — Download

👉 **[Click here to download the latest version](https://github.com/rsk2111999/linkedin-easy-apply-bot/releases/latest/download/linkedin-easy-apply-extension.zip)**

Unzip the file. You'll get a folder called `linkedin-easy-apply-extension`.

---

### Step 2 — Load into Chrome

1. Open Chrome and go to: **chrome://extensions**
2. Turn on **Developer mode** (toggle in the top-right corner)
3. Click **Load unpacked**
4. Select the `linkedin-easy-apply-extension` folder you just unzipped

The extension icon (⚡) will appear in your Chrome toolbar.

> **Firefox / Edge users:** This works on any Chromium browser (Edge, Brave, Arc). Same steps.

---

### Step 3 — Fill your profile

Click the ⚡ icon → fill in your details across the **Profile** and **Job Prefs** tabs → click **Save Profile**.

You only do this once.

---

## How to use it

1. Go to [LinkedIn Jobs](https://www.linkedin.com/jobs/) and search for roles you want
2. Click the ⚡ icon → go to the **Filters** tab → set which job titles to include/exclude
3. Go to the **Run** tab → click **⚡ Batch Apply All**
4. Watch the bot work — it applies to every matching Easy Apply job on the page automatically

That's it. No more inputs needed.

## Job Filters

Set these in the **Filters** tab so the bot only applies to relevant jobs:

- **Job Title Keywords** — e.g. `Product Manager, PM, APM` — the job title must contain at least one
- **Exclude Keywords** — e.g. `Senior, Director, VP` — skips jobs with these words
- **Remote Only** — only applies to jobs tagged Remote

## FAQ

**Will it apply to jobs I've already applied to?**
No. It tracks every job ID it applies to and skips duplicates.

**What if a question comes up it doesn't know how to answer?**
It leaves the field blank and moves on. You'll see it in the Activity Log.

**Is this safe to use?**
It behaves like a human — types character by character, moves the mouse, adds random pauses. It doesn't scrape or spam. Use responsibly.

**Can I stop mid-way?**
Yes — click **Stop** in the Run tab at any time.

**How do I update my profile?**
Click the ⚡ icon → edit fields → Save Profile. Changes take effect immediately.
